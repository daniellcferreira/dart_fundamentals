# Calculadora em Dart

![Dart](https://img.shields.io/badge/Dart-Linguagem%20de%20Programa%C3%A7%C3%A3o-0175C2?style=flat-square&logo=dart)

## Descrição
Este projeto consiste em uma calculadora desenvolvida em Dart que opera exclusivamente pelo terminal.  
Ela foi criada com o objetivo de demonstrar a leitura de entradas do usuário, o uso de estruturas de decisão e a separação de responsabilidades por meio de funções.  
Ao iniciar o programa, o usuário é orientado a informar um número inicial, seguido pela escolha de uma operação matemática válida.  
O sistema valida a operação digitada e, caso o operador informado não esteja entre os permitidos, solicita novamente até que uma opção válida seja fornecida.  

Após a validação da operação, o usuário deve inserir o segundo número que fará parte do cálculo.  
O programa então utiliza funções específicas para cada tipo de operação, como soma, subtração, multiplicação e divisão, garantindo uma execução organizada e clara.  

A lógica de cálculo é centralizada em um método que direciona a operação correta com base no operador informado.  
Com isso, o código se mantém modular, simples de entender e fácil de estender para futuras funcionalidades, como novas operações ou melhorias na interação com o usuário.  

Ao final, o resultado da operação escolhida é exibido no console de forma direta e objetiva.  
Este projeto é ideal para quem está começando com Dart e deseja entender como manipular entrada e saída de dados, trabalhar com funções e aplicar lógica básica de programação em um ambiente de linha de comando.
