# Dart Fundamentals


[![Dart](https://img.shields.io/badge/Dart-Programming_Language-%230175C2?logo=dart&logoColor=white)](https://dart.dev)

## Descrição

O repositório `dart_fundamentals` foi criado para demonstrar fundamentos essenciais da linguagem Dart. Esta linguagem, desenvolvida pelo Google, é amplamente utilizada para o desenvolvimento de aplicações multiplataforma, como aplicativos móveis com o framework Flutter.

Dart é uma linguagem moderna, orientada a objetos, fortemente tipada, com foco na simplicidade, desempenho e escalabilidade. Este repositório pretende ser um ponto de aprendizado prático e teórico sobre os principais conceitos utilizados por desenvolvedores ao trabalhar com Dart.

O objetivo principal deste repositório é proporcionar exemplos e demonstrações de conceitos fundamentais da linguagem, permitindo que novos desenvolvedores ou aqueles que estão migrando para Dart possam entender sua estrutura e funcionalidades básicas.

## Funcionalidades Abordadas

O projeto cobre vários tópicos sobre Dart, incluindo:

- Declaração de variáveis e tipos de dados.
- Funções, métodos e parâmetros.
- Estruturas de controle, como loops e condicionais.
- Orientação a objetos (Classes, Objetos, Herança e Polimorfismo).
- Manipulação de coleções (Listas, Conjuntos e Mapas).
- Tratamento de erros e exceções.
- Modificação de dados com métodos utilitários.
- Null Safety: Recurso para evitar erros ao acessar valores nulos.
- Programação Assíncrona com `Future` e `Stream`.

Todos os exemplos incluídos ao longo do projeto possuem código comentado e explicativo para maximizar o aprendizado.

## Por que este repositório é útil?

- **Para iniciantes:** Um ponto de partida para entender os conceitos primários de Dart.
- **Refinamento de conhecimento:** Serve como material de consulta para quem deseja aprofundar no desenvolvimento Dart.
- **Bases sólidas:** Introdução a uma das linguagens mais utilizadas em aplicações modernas.
