## Descrição
AnyBank é um projeto simples desenvolvido em Dart para simular operações bancárias utilizando conceitos de orientação a objetos.  
O sistema permite criar diferentes tipos de contas e realizar operações como enviar valores, receber depósitos e calcular rendimentos.  
O objetivo é demonstrar herança, polimorfismo e encapsulamento em um ambiente didático.

## Funcionalidades
- Criação de contas bancárias com comportamentos diferentes:
  - Conta Corrente
  - Conta Poupança
  - Conta Empresa
  - Conta Investimento
- Exibição de saldo da conta.
- Envio de valores com validação de saldo.
- Recebimento de depósitos.
- Cálculo de rendimento para contas com essa funcionalidade.
- Organização modular com arquivos separados para melhor manutenção.
