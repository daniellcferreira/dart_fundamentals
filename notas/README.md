## Descrição
Projeto desenvolvido em Dart para gerenciar notas via terminal. O usuário pode adicionar novas notas e listar todas as notas cadastradas durante a execução. O sistema utiliza entrada padrão, listas e controle de fluxo para organizar as funções.

## Funcionalidade
- Menu interativo no terminal
- Adicionar novas notas
- Listar notas cadastradas
- Validação de entradas do usuário
